Software Engineer

  1. ALL JOBS
  2. SOFTWARE ENGINEER/IT STAFF

Proficiencies:
HTML CSS3 JavaScript .NET MVC JAVA MySQL Other Related Technologies

Responsibilities

Implement design, installation, testing and maintenance of software systems.

Responsible for writing and coding individual programs or providing an entirely new software.

Review current system and propose new ideas for system improvement or upgrading.

Work closely with various business segments to achieve needed data analysis for reporting needs.

Apply coding bes practices and perform code reviews.

Conduct software testing during the development phase to ensure that it will run accurately.

Prepare modules and train users on how to use the system and help them understand the process.

Perform system maintenance once the software is up and running.

Update superiors of work progress; give feedback on work performed, difficulties encountered and issues identified.

Accomplish organization mission by completing related results as needed.

Adhere to the profession's code of ethics.


Qualifications

Candidate must possess at least a Bachelor's/College Degree in Computer Science or Information Technology.

Strong analytical and problem-solving skills.

Proficient in oral and written communication skills.

Must be willing to work overtime when necessary.

Driven to make highly customized applications.

An optimist and brings solutions.

Inquisitive, insightful, flexible, fast-learner and attentive to detail.

Motivated, curious, and passionate about building products that make an impact.

Proactive, independent and able to work productively with minimal supervision.

Understands a small firm culture and knows things can be chaotic and change quickly.

Fresh graduates and entry level applicants are preferred and encouraged to apply.


Position: Full-Time

Working Hours: Regular Hours (Monday-Friday)
